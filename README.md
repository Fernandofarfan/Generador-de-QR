# Generador de QR

Web app para crear QRs fácil. 

Funciona para:
* Links
* WiFi
* Contactos
* Texto simple

También escanea QRs usando la cámara.

## Uso
Solo abrir `index.html` o entrar a la web. Todo funciona offline si la instalás.

Se pueden cambiar colores y poner logos.

> Repo simple en JS.

## Dudas frecuentes
**¿Vencen los QR?**
No, son para siempre. Mientras el link que pusiste funcione, el QR va a funcionar. No tienen fecha de vencimiento.
